# MySQL User Manager

Simple Python CLI application that connects to a MySQL database running in Docker.

## Features

- List users from database
- Add new users
- Connect Python application to MySQL running in Docker

## Technologies

- Python
- MySQL
- Docker
- PyCharm

## Project structure

- `main.py` - application logic and CLI menu
- `db.py` - database connection
- `requirements.txt` - Python dependencies

## Requirements

- Python 3
- Docker
- MySQL Docker image
- mysql-connector-python

## Install dependencies

```bash
pip install -r requirements.txt
